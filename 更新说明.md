## 📋 更新内容

### 1️⃣ 自动启动服务器功能

以下3个测试脚本现在会**自动检查并启动服务器**:

```bash
bash quick_test.sh                # 快速测试
bash start_and_test.sh            # 启动+测试
bash week_automated_test_final.sh # 7天测试
```

**工作原理**:
- ✅ 自动检查服务器是否运行
- ✅ 如果未运行,自动后台启动
- ✅ 等待15秒确保启动成功
- ✅ 验证服务器健康状态
- ✅ 然后开始测试
