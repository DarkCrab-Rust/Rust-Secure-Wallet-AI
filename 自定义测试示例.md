## 📊 计算器

### 快速计算测试轮数
```
测试轮数 = 测试时长(分钟) / 测试间隔(分钟)

示例:
- 2天 + 6小时 = 3240分钟
- 间隔30分钟
- 轮数 = 3240 / 30 = 108轮
```
